Défi 1 : L'ouverture du coffre numérique

Le joueur commence devant un coffre numérique verrouillé. Pour y accéder, il doit se connecter en contournant le formulaire de connexion.

Ce formulaire est vulnérable à une attaque par injection SQL.



Objectif: Manipuler le formulaire de connexion pour insérer une requête SQL malveillante et obtenir le code caché dans la base de données.

Mécanisme: Une fois le code découvert, collez-le dans l'URL de la page suivante (par exemple page2.php?code=B7R8J2D) pour accéder au défi suivant.
